# Monocular Depth

Monocular Depth is a browser-based artwork where space, language, and perception drift out of alignment.

A dark field opens.

Depth appears through motion, scale, opacity, parallax, and occlusion.

Sometimes language surfaces.

Then it disappears again.

Deployed with GitHub Pages.

## Local

```sh
python3 -m http.server 8000
```

Open `http://localhost:8000`.

## Notes

The work runs entirely in the browser.

There is no backend, no analytics, no tracking, and no build process.

GitHub Pages can serve the repository root directly.
